# pixel_setup — TikTok Ads Split Testing

A simple static site introducing **split testing on TikTok Ads Manager**, built by the Split Test team.

Content is based on the official TikTok Ads Manager help docs:
- [Split testing overview](https://ads.tiktok.com/help/article/split-testing?lang=en)
- [Create a split test](https://ads.tiktok.com/help/article/create-split-test?lang=en)
- [TikTok For Business](https://ads.tiktok.com/business/en-US)

## Local preview

```bash
python3 -m http.server 8000
# open http://localhost:8000
```

## Enable GitHub Pages

Repo Settings -> Pages -> Source: "Deploy from a branch" -> Branch: `main` / `(root)`.
