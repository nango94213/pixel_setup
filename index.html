What is a split test?

Split testing in TikTok Ads Manager lets you test two different versions of your ads to determine which one performs best. Your audience is divided equally, and each group sees only one version — so the comparison is clean, with no overlap.

⚖️

Audience isolation

Each audience group exclusively sees one ad group, so your test versions never compete against each other.

📊

Statistical reliability

A scientifically designed model uses a 90% confidence rate to declare a winner — results are validated only when significant.

🚀

Scale the winner

When the test ends, keep running the winning ad group and launch follow-up tests to keep refining.

How to create one

From the TikTok Ads Manager, it's a guided flow across three tabs.

  1. 1

    Start a campaign

    In Ads Manager, open the Campaign tab, click Create, pick an objective, then toggle on Create a split test.

  2. 2

    Configure the Ad Group

    Set your control group settings and schedule the test for at least 7 days — enough time for the system to collect a useful sample.

  3. 3

    Build the Ad

    Complete the ad settings for the control group. It serves automatically on its scheduled start date.

  4. 4

    Set up the Split Test tab

    Choose your split test variable and a Key Metric — the system uses the Key Metric to compare the two ad groups and determine the highest-performing ad. Set up the test ad group and click Complete.

Frequently asked

Quick answers before you launch your first test.

Do both versions really get a fair shot?+

Yes — your audience is divided evenly and each group exclusively sees one ad group, so the two versions never compete for the same impression.

How long should I let a test run?+

Schedule at least 7 days. Shorter windows rarely collect enough samples for the system to reach a statistically significant verdict.

What happens when a winner is declared?+

Keep the winning ad group running, pause the loser, and queue up a follow-up test — iterate on the winner instead of starting from scratch.

Can I test more than one variable at once?+

Use the Custom (multi-variable) category for that — but for the cleanest read, test one variable at a time so you know exactly what moved the metric.
